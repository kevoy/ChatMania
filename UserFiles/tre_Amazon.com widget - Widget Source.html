Widget Source

Why use Widget Source?
  • Create multiple widgets easily
  • Build cool mashups with other web services
  • Integrate widgets with your web site experience

What is Widget Source?

Widget Source is a new way to create Amazon Widgets in-code.

Until now, the only way for you to create a widget was by using our WYSIWYG widget editor in Associates Central. If you wanted to create a large number of widgets, it was tedious to build each widget by hand.

Using Widget Source, you can now create widgets programmatically and embed them anywhere on your site. Widget Source gives you complete programmatic control over the widget look-and-feel and provides you with multiple ways to add products to the widget. Learn more
